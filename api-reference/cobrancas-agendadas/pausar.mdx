title: "Pausar Cobrança"

---

## Visão Geral

Pausa a cobrança. Nenhum e-mail (cliente ou time) é enviado até você retomá-la com [`/resume`](/api-reference/cobrancas-agendadas/retomar).

**Permitido a partir de:** `scheduled` / `due_today` / `overdue`.

## Exemplo de Requisição

<CodeGroup>

```bash cURL
curl -X POST https://garu.com.br/api/scheduled-charges/sch_abc123/pause \
  -H "Authorization: Bearer sk_test_sua_chave" \
  -H "Content-Type: application/json" \
  -d '{ "reason": "em negociação" }'
```

```javascript JavaScript
import { Garu } from '@garuhq/node';

const garu = new Garu({ apiKey: process.env.GARU_API_KEY });

await garu.scheduledCharges.pause('sch_abc123', { reason: 'em negociação' });
```

```python Python
import requests
import os

requests.post(
    "https://garu.com.br/api/scheduled-charges/sch_abc123/pause",
    headers={"Authorization": f"Bearer {os.environ['GARU_API_KEY']}"},
    json={"reason": "em negociação"}
)
```

</CodeGroup>

## Parâmetros

<ParamField path="id" type="string" required>
  ID da cobrança (`sch_…`).
</ParamField>

<ParamField body="reason" type="string">
  Texto livre persistido no log de auditoria.
</ParamField>

## Resposta

A cobrança atualizada com `status: "paused"`. Um evento `paused` é apendado e o webhook `scheduled_charge.paused` é disparado.
