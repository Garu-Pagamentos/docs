## Visão Geral

Retorna os detalhes completos de um preço de assinatura específico pelo ID.

## Exemplo de Requisição

<CodeGroup>

```bash cURL
curl -X GET https://api.garu.com.br/api/subscription-prices/456 \
  -H "Authorization: Bearer sk_test_sua_chave"
```

```javascript JavaScript
const response = await fetch(
  'https://api.garu.com.br/api/subscription-prices/456',
  {
    headers: {
      'Authorization': `Bearer ${process.env.GARU_API_KEY}`
    }
  }
);

const preco = await response.json();
console.log(`Plano: ${preco.name}`);
console.log(`Valor: R$ ${(preco.amount / 100).toFixed(2)}`);
console.log(`Intervalo: ${preco.billingInterval}`);
```

```python Python
import requests
import os

url = "https://api.garu.com.br/api/subscription-prices/456"
headers = {
    "Authorization": f"Bearer {os.environ['GARU_API_KEY']}"
}

response = requests.get(url, headers=headers)
preco = response.json()

print(f"Plano: {preco['name']}")
print(f"Valor: R$ {preco['amount'] / 100:.2f}")
print(f"Intervalo: {preco['billingInterval']}")
```

</CodeGroup>

## Resposta de Sucesso

<ResponseField name="id" type="number">
  ID interno do preço.
</ResponseField>

<ResponseField name="uuid" type="string">
  Identificador único usado no link de checkout.
</ResponseField>

<ResponseField name="productId" type="number">
  ID do produto pai.
</ResponseField>

<ResponseField name="name" type="string">
  Nome do plano.
</ResponseField>

<ResponseField name="amount" type="number">
  Valor em centavos.
</ResponseField>

<ResponseField name="billingInterval" type="string">
  Intervalo de cobrança: `weekly`, `monthly`, `quarterly`, `annually`.
</ResponseField>

<ResponseField name="trialDays" type="number">
  Dias de teste gratuito.
</ResponseField>

<ResponseField name="isActive" type="boolean">
  Se o plano está ativo para novas assinaturas.
</ResponseField>

{
