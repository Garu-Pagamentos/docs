## Visão Geral

Use este endpoint para atualizar as informações de um produto existente. Apenas os campos enviados serão atualizados.

## Parâmetros de Path

<ParamField path="id" type="number" required>
  ID do produto a ser atualizado.
</ParamField>

## Parâmetros do Body

Todos os campos são opcionais. Envie apenas os que deseja atualizar.

<ParamField body="name" type="string">
  Novo nome do produto.
</ParamField>

<ParamField body="description" type="string">
  Nova descrição do produto.
</ParamField>

<ParamField body="value" type="number">
  Novo preço do produto (mínimo R$ 5,00).
</ParamField>

<ParamField body="pix" type="boolean">
  Habilitar/desabilitar pagamento via PIX.
</ParamField>

<ParamField body="creditCard" type="boolean">
  Habilitar/desabilitar pagamento via cartão de crédito.
</ParamField>

<ParamField body="boleto" type="boolean">
  Habilitar/desabilitar pagamento via boleto.
</ParamField>

<ParamField body="installments" type="number">
  Novo número máximo de parcelas (1-12).
</ParamField>

<ParamField body="image" type="string">
  Nova URL da imagem do produto.
</ParamField>

<ParamField body="returnUrl" type="string">
  Nova URL de retorno após pagamento.
</ParamField>

<ParamField body="returnUrlButtonText" type="string">
  Novo texto do botão de retorno.
</ParamField>

## Exemplo de Requisição

<CodeGroup>

```bash cURL
curl -X PATCH https://api.garu.com.br/api/products/123 \
  -H "Authorization: Bearer sk_test_sua_chave_api" \
  -H "Content-Type: application/json" \
  -d '{
    "value": 347.00,
    "installments": 6
  }'
```

```javascript JavaScript
const response = await fetch('https://api.garu.com.br/api/products/123', {
  method: 'PATCH',
  headers: {
    'Authorization': `Bearer ${process.env.GARU_API_KEY}`,
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    value: 347.00,
    installments: 6
  })
});

const product = await response.json();
console.log(`Produto atualizado: ${product.name} - R$ ${product.value}`);
```

```python Python
import requests
import os

url = "https://api.garu.com.br/api/products/123"
headers = {
    "Authorization": f"Bearer {os.environ['GARU_API_KEY']}",
    "Content-Type": "application/json"
}
payload = {
    "value": 347.00,
    "installments": 6
}

response = requests.patch(url, json=payload, headers=headers)
product = response.json()
print(f"Produto atualizado: {product['name']} - R$ {product['value']}")
```

</CodeGroup>

<Note>
  O `uuid` do produto **não muda** ao atualizar. O link de pagamento permanece o mesmo.
</Note>

## Erros Comuns

<AccordionGroup>
  <Accordion title="404 - Produto não encontrado">
    ```json
    {
      "statusCode": 404,
      "message": "Product not found"
    }
    ```
    **Solução:** Verifique se o ID do produto está correto.
  </Accordion>

  <Accordion title="403 - Sem permissão">
    ```json
    {
      "statusCode": 403,
      "message": "Insufficient permissions"
    }
    ```
    **Solução:** Você só pode atualizar produtos criados com sua conta.
  </Accordion>
</AccordionGroup>
