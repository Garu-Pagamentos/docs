## Aceite pagamentos com Claude Code

Conecte a Garu ao Claude Code e crie cobranças, gerencie clientes e consulte transações diretamente pelo terminal.

<Steps>

### Instale o MCP Server

```bash
claude mcp add garu -- npx -y --package=@garuhq/mcp@latest garu-mcp
```

Depois, defina a variável de ambiente `GARU_API_KEY` no seu projeto ou shell:

```bash
export GARU_API_KEY=YOUR_API_KEY
```

Substitua `YOUR_API_KEY` pela sua chave de API. Veja como obter em [Autenticação](/api-reference/autenticacao).

### Instale as Skills

```bash
npx skills add Garu-Pagamentos/garu-skills
```

As skills ensinam o Claude a seguir boas práticas de pagamento (LGPD, PCI-DSS, tratamento de erros).

### Comece a usar

Peça ao Claude para executar tarefas de pagamento. Experimente:

```
Crie uma cobrança PIX de R$49,90 para joao@email.com
```

</Steps>
