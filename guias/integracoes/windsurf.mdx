## Aceite pagamentos com Windsurf

Conecte a Garu ao Windsurf e deixe o agente criar cobranças, consultar transações e gerenciar clientes enquanto você codifica.

<Steps>

### Configure o MCP Server

Crie ou edite o arquivo `.windsurf/mcp.json` na raiz do seu projeto:

```json
{
  "mcpServers": {
    "garu": {
      "command": "npx",
      "args": ["-y", "@garuhq/mcp"],
      "env": {
        "GARU_API_KEY": "YOUR_API_KEY"
      }
    }
  }
}
```

Substitua `YOUR_API_KEY` pela sua chave de API. Veja como obter em [Autenticação](/api-reference/autenticacao).

### Instale as Skills

```bash
npx skills add Garu-Pagamentos/garu-skills
```

### Reinicie o Windsurf

Feche e reabra o Windsurf para carregar o servidor MCP.

### Comece a usar

Peça ao agente para executar tarefas de pagamento. Experimente:

```
Crie uma cobrança PIX de R$49,90 para joao@email.com
```

</Steps>
