## Aceite pagamentos com Bolt

Use a Garu no Bolt para adicionar pagamentos PIX, Cartão e Boleto ao seu app com prompts simples.

<Steps>

### Configure a variável de ambiente

No Bolt, vá em **Settings → Environment Variables** e adicione:

| Variável       | Valor                                                        |
| -------------- | ------------------------------------------------------------ |
| `GARU_API_KEY` | Sua chave de API ([como obter](/api-reference/autenticacao)) |

### Instale a SDK

Peça ao Bolt para instalar a dependência:

```
Instale o pacote @garuhq/node
```

### Instale as Skills

<Snippet file="snippets/skills-install.mdx" />

### Peça para criar a integração

Use prompts em português para descrever o que você precisa. Experimente:

```
Adicione pagamento PIX de R$49,90 usando @garuhq/node.
A chave de API está em process.env.GARU_API_KEY.
```

</Steps>

## Mais Exemplos de Prompts

Crie uma página de checkout com PIX e cartão de crédito.
